# Yandex Metrika setup

## 1. Создание счетчика

1. Перейдите в [Яндекс Метрику](https://metrika.yandex.ru/).
2. Нажмите `Добавить счетчик`.
3. Укажите домен сайта и включите Webvisor.

## 2. Где взять ID

После создания счетчика откройте его настройки.
`ID счетчика` отображается вверху (число вида `12345678`).

## 3. Куда вставить ID

В файле `.env.local` в корне проекта:

```env
NEXT_PUBLIC_YANDEX_METRIKA_ID=12345678
```

## 4. Как проверить, что подключилось

1. Запустите сайт (`npm run dev`).
2. Откройте DevTools -> Network и убедитесь, что есть запросы к `mc.yandex.ru`.
3. В интерфейсе Метрики проверьте `Визиты в реальном времени`.

## 5. Какие цели уже отправляются

- `submit_lead`
- `click_phone`
- `click_whatsapp`
- `click_catalog`
- `open_product`
- `scroll_50`
- `scroll_90`

Цели можно посмотреть в Метрике: `Настройка -> Цели`.
